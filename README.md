# BMW Global Sales & EV Performance (2010–2024)📈

End-to-end BI portfolio project built with Excel → SQL → Power BI.

## Dashboard
![Dashboard preview](assets/dashboard-screenshot.jpg)

## Data
- Main dataset (Kaggle): https://www.kaggle.com/datasets/ayeshaseherr/bmw-dataset
- Processed CSVs generated with SQL (included in `/data/processed`)

## KPIs / Questions answered
- Revenue by region (Q1)
- Top models by revenue (Q2)
- EV sales trend by year (Q3)

## How to reproduce
1. Download the dataset from Kaggle (link above)
2. Run the SQL scripts in `/sql`
3. Load the processed CSV outputs into Power BI (or refresh the model)
